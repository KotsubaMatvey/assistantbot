Assistant Bot

Функции и возможности бота

Этот справочник собран по текущему коду проекта. Бот работает как персональный second brain: ведёт диалог в чате, сохраняет данные по явному намерению, помогает с задачами, источниками, бюджетом, домом, рынками, покупками и дает операторские команды для контроля сессий.

Основные возможности

Second brain память

Обычный текст используется для диалога и сам не сохраняется. Мысли, задачи, решения, ссылки, людей, предпочтения, важные пины и голосовые расшифровки можно сохранять явными фразами или командами.

Markdown vault SQLite/FTS индекс Spaces

Поиск и контекст

Бот ищет заметки, показывает сегодняшнюю ленту, открытые задачи, недавние записи, коллекции, источники и отвечает на вопросы по локальной памяти.

/memory /ask /context

Операторский контроль

Есть отдельные команды для новой логической сессии, сжатия контекста, статуса, режима ассистента, trace-режима, подробности ответов и выбора помощника.

/status /new /compact

Задачи и автоматизации

Бот ведет задачи, напоминания, отложенные follow-up, периодические jobs, утренний briefing, вечерний review и недельный план.

/agenda /job_add /morning

Источники и обучение

Можно добавлять RSS, GitHub и URL-источники, синхронизировать их в память, читать RSS-дайджесты, сохранять веб-страницы и смотреть доверие к источникам.

rss github url

Дом, бюджет и семья

Поддерживаются домашний склад, списание продуктов, план докупки, чеки, бюджет месяца, доходы, расходы, счета, подписки, cashflow и семейный общий список.

/pantry /budget /family

Рынки

Бот получает BTC, BTC.D и основные мировые индексы, формирует market watch, market brief, sentiment, risk regime и сохраняет рыночные заметки в память.

BTC BTC.D индексы

Покупки и цены

Shopping skill сравнивает корзину по магазинам, помнит последнюю корзину, учитывает настройки магазинов, следит за ценовыми сигналами и подсказывает выгодные докупки.

Smart Магнит SPAR 5ka Fix Price

Безопасные локальные действия

Локальные tools, удаление заметок через подтверждение, audit log, access allowlist, pairing-коды и админ-диагностика отделены от обычного пользовательского потока.

/tools /approve audit

Telegram Mini App

Mini App открывается кнопкой над полем ввода Telegram или командой /mini_app, если задан TG_MINI_APP_URL. В проекте это Vite + React + TypeScript интерфейс из папки miniapp/.

Today

Показывает agenda, дайджест дня, задачи, напоминания, заметки и фокусные элементы.

Assistant

Пульт для команд ассистента: status, compact, new, agenda, today, tasks, context и похожие действия.

Finance

Читает и обновляет счета, расходы, доходы, подписки, чеки, бюджет и прогноз cashflow.

Memory

Показывает здоровье памяти, объекты, источники, события и позволяет добавлять, удалять и синхронизировать источники.

Shopping

Отправляет корзину в бот для сравнения цен и поддерживает быстрые shopping-сценарии.

Markets

Обновляет рыночные данные через backend API и показывает текущие сигналы.

Ограничения и условия работы

Локальная память

Основное хранилище находится в OBSIDIAN_VAULT_PATH. Бот сохраняет Markdown, локальные объекты, индексы и историю действий.

Внешняя сеть

При включённом LLM свободный диалог может отправляться настроенному cloud provider. Сеть также используется для /learn_url, RSS, market data, scraping и обновления цен.

Цены не гарантированы

Публичные данные магазинов могут отличаться из-за региона, наличия, карты лояльности, акций и времени обновления.
